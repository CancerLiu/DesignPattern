因为代理模式比较重要，这里单独列出一个说明文档进行整理总结——
(1)常用的几种代理模式
①远程代理(Remote Proxy)——RCP
②虚拟代理(Virtual Proxy)——懒加载
③保护代理(Protect Proxy)——正向代理
④缓冲代理(Cache Proxy)——反向代理
⑤智能引用代理(Smart Reference Proxy)——切面编程

(2)代理模式与装饰模式的区别
①代理模式主要是给真实主题类增加一些全新的职责，这些职责与原有职责不属于同一个问题域;
  装饰模式是通过装饰类为具体构件增加一些相关的职责，是对原有职责的扩展，和原有职责属于同一个问题域。
②代理模式主要是为了控制对对象的访问;装饰模式主要是为对象动态地增加功能。
